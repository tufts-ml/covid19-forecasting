# covid19-forecasting plots

A simple Plotly Dash application that displays plots for various stages of hospitalization

PI: Michael C. Hughes

## Usage

Configurations are stored in plot/config.json

To run the app:

```
$ cd plot
$ python app.py
```

Visit: 
[http://127.0.0.1:8050/](http://127.0.0.1:8050/)
